# VSA Demo with Jones Constant

## Описание

Демо-композиция Vector Symbolic Architecture (VSA) с использованием константы Джонса (φ = 1.618...) из SU(2)₃ Черн-Симонс теории.

## Используемые модули

```t27
use base::types;       // Trit enum: {neg = -1, zero = 0, pos = 1}
use math::constants;      // mat::abs(x), PI, E
use vsa::ops;          // dot_product(), VSA_DIM = 1024
use physics::su2_chern_simons;  // jones_polynomial_at_5th_root() → φ
use math::sacred_physics;  // PHI = 1.618...
```

## Функциональность

### JonesSignature (структура)
Компактное представление топологической сложности структуры:
- `jones_value`: Значение полинома Джонса (теоретически φ)
- `dot_product`: Скалярное произведение с эталонной структурой
- `complexity_level`: Уровень сложности (0=LOW, 1=MID, 2=HIGH)

### Классификация
Структуры классифицируются на основе:
1. Сходства с константой φ (через Jones polynomial)
2. Скалярного произведения с эталонной структурой
3. Пороговые значения PHI_THRESHOLD_LOW = 0.5 и PHI_THRESHOLD_HIGH = 2.0

## Примечание

Это демо-композиция для проверки гипотезы о топологических ускорителях. Не производит реальной топологической классификации — вместо этого использует константу φ и скалярные произведения как эвристику для разделения структур на уровни сложности.

## Источник

Полином Джонса при 5-м корне единства вычисляется как φ в рамках SU(2)₃ Черн-Симонс теории (уровень k=3).
