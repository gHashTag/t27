# TRIOS Hybrid

HybridBigInt — Memory-efficient ternary number representation.

## Features

- BigInt operations for ternary numbers
- Packed trit representation
- Hybrid number system combining binary and ternary

## Status

- ✅ Phase 1: Core primitive extracted from monolith
- 📋 Phase 2: FFI bindings to Zig implementation
